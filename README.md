# 삼가 고인의 명복을 빕니다 — 완벽한 추모글이 30개였는데

> 2026 KAIST AI × 실패 아이디어 공모전 「미래에서 온 오답노트」 출품작 보완 자료

2036년 12월, 경기도 수원시의 한 장례식장. 고인의 딸은 밤새 조문객들의 추모글 30개를 읽었다. 문장은 아름다웠고 위로는 완벽했지만, 어디에도 아버지가 없었다. 조문객 30명이 각자 AI에게 추모글을 맡겼기 때문이다. 문장은 달랐지만 구조는 같았다. 감정은 전달됐지만, 사람은 사라졌다.

이 저장소는 그 실패를 미리 목격하고 기록한 사전 실패 분석(Pre-Mortem)의 보완 자료입니다.

## 3단계 요약

**예견된 실패** : AI가 감정 표현의 기본값이 된 2036년. 추모·사과·이별·축하의 언어는 완벽해졌지만, 그 안에서 사람이 사라졌다.

**원인 진단** : 씨앗은 2026년에 심어졌다. 위로와 감사의 말이 어려울 때마다 사람들은 그 불편함을 AI에게 맡겼고(감정 표현의 외주화), 창의적 판단도 AI에게 먼저 물었다(사고의 외주화). 그 불편함이 사실은 인간이 성장하고 서로에게 닿는 유일한 방법이라는 신호를 놓쳤다.

**대응 방안**  AI 금지가 아니라 '날 것'을 보존하는 세 겹의 장치. ① 개인: AI 없이 직접 쓴 기록만 모으는 날 것의 아카이브 ② 교육부: 초·중·고 주 1회 'AI 프리존 표현 시간' ③ 사회: 감정 출처 자율 표기 문화와 'Analog Zone' 캠페인.

## 부고장 데모 보는 법

HTML 파일을 브라우저에서 열면 됩니다(링크는 구글폼에 삽입). 별도 설치나 서버가 필요 없습니다.

화면 속 온라인 추모관에는 추모글 30개가 올라와 있습니다. **"문장을 겹쳐 보기"** 버튼을 누르면 30개의 글이 전부 `① 애도 인사 → ② 고인 회상 → ③ 유가족 위로 → ④ 마무리 기원`이라는 동일한 뼈대로 쓰였다는 것이 색으로 드러납니다.

한 가지 고백:  이 추모글 30개는 사람이 쓴 것이 아닙니다. 네 개의 문장 묶음에서 하나씩 뽑아 조합하는 몇 줄의 코드가 만들었습니다. 표현은 제각각이지만 구조는 단 하나 — 이 데모의 코드 자체가 'AI 평균화'의 재현입니다.
